# Demo Video Script — Claude Companion

**الهدف**: فيديو واحد (60–90 ثانية) يُظهر القيمة الفوريّة في أوّل 5 ثوانٍ، ثمّ مهمّة واقعيّة مُذهلة، ثمّ CTA.

**قاعدة ذهبيّة**: الـ 3 ثوانٍ الأولى يجب أن تُظهر **Claude يتحرّك داخل صفحة فعلاً** — ليس شعار، ليس intro، ليس voiceover. **حركة حقيقيّة**.

---

## 🎬 النسخة العربيّة (90 ثانية)

### Shot 1 — [0:00 – 0:04] ← الـ hook
**مرئيّ**: Browser مفتوح على Google. side panel مفتوح. المستخدم يضغط زرّ الميكروفون ويقول بصوته:
> *"افتح يوتيوب ودوّر على lo-fi beats، وشغّل أوّل فيديو."*

**تأثير**: ترى Side Panel يظهر "⚙ جارٍ..."، ثمّ يتحرّك المتصفّح تلقائيّاً — يفتح تبويب يوتيوب، يكتب في صندوق البحث، يضغط بحث، يختار أوّل نتيجة، الفيديو يبدأ بالعمل.

**نصّ الشاشة** (bottom third): `Claude Companion · يعمل باشتراك Claude Max`

---

### Shot 2 — [0:04 – 0:12] ← **القيمة الرئيسيّة**
**صوت**:
> *"مساعد متصفّح يعمل باشتراك Claude Max — **بدون مفاتيح API، بدون خادم خارجيّ، بدون رسوم إضافيّة**."*

**مرئيّ**: overlay بسيط يُظهر 3 أيقونات:
- 🔒 Local-first
- 💸 Zero API spend
- 🌍 عربيّ + إنجليزيّ

---

### Shot 3 — [0:12 – 0:35] ← **المهمّة المُركّبة**
**السيناريو المقترَح** (اختر واحداً فقط حسب جمهورك):

#### الخيار A — للمطوّرين (HN/X):
> *"اقرأ أوّل 3 نتائج على Hacker News، لخّصها لي في جدول، وانسخها للحافظة."*

ترى Claude:
1. ينتقل إلى news.ycombinator.com
2. يفتح 3 مقالات في تبويبات (أو يقرأ العناوين + روابط)
3. يرجع للـ side panel ويُنشئ جدول markdown
4. Side panel يُظهر زر "نسخ" → ضغطة → ✅

#### الخيار B — للعرب (LinkedIn/X MENA):
> *"ادخل نون، ابحث عن iPhone 15 Pro، وقارن الأسعار مع الإمارات."*

ترى Claude يتنقّل بين نون.كوم و noon.com/ae، يقرأ الأسعار، يُنشئ مقارنة.

#### الخيار C — للقوّة الخالصة (ProductHunt):
> *"لخّص لي هذه الصفحة الطويلة وترجمها للعربيّة."*

صفحة Wikipedia طويلة → ملخّص عربيّ بالـ markdown كامل في Side Panel.

**💡 توصيتي**: استخدم الخيار B — لا أحد آخر يفعلها بالعربيّة. ميّزة قاتلة.

---

### Shot 4 — [0:35 – 0:50] ← **الاستعراض السريع**
**صوت**:
> *"18 أداة متصفّح: قراءة، ضغط، كتابة، screenshot، وأكثر. كلّها محلّيّة. كلّها آمنة. MIT مفتوحة المصدر."*

**مرئيّ** — 4 لقطات سريعة كل 2 ثانية:

- Settings: memory + tasks chips
- Voice input بالعربيّة
- Toolbar icon على Brave + Chrome + Edge (نفس الإضافة)

### Shot 5 — [0:50 – 1:20] ← **CTA**

## 🎙️ Voiceover — قواعد التسجيل

1. **صوت إنسان حقيقيّ، ليس TTS**. الفيديوهات الـ TTS = فشل محقّق في 2026.
2. **العربيّة أولاً**. جمهورك الفريد MENA. سجّل العربيّ بنفسك، ثمّ أتبِعه بترجمة إنجليزيّة (أو نسخة إنجليزيّة منفصلة).
3. **سرعة الكلام**: أسرع 15% من معدّلك الطبيعيّ. الانتباه على YouTube/X قصير.
4. **لا موسيقى صاخبة** — مجرّد ambient خفيف.
5. **سجّل الشاشة بـ 60fps** لأنّ حركة الـ automation هي البطلة.

## 🧪 اختبار الفيديو قبل النشر

قبل أن تنشر، أرسله لـ **3 أشخاص**:
- [ ] شخص لا يعرف Claude أصلاً → هل فهم القيمة في أوّل 10 ثوانٍ؟
- [ ] شخص مطوّر → هل بدا المنتج تقنيّاً مُقنعاً؟
- [ ] شخص عربيّ → هل النطق/الترجمة طبيعيّان؟

إذا 2 من 3 قال "أريد أجرّبه" → جاهز للنشر.
إذا لا → عُد وعدّل.
