# Launch Kit — Claude Companion

ملفّات جاهزة لإطلاق Claude Companion للعامّة.

## الترتيب الصحيح

1. **اقرأ `03-launch-checklist.md` أوّلاً** — يعطيك نظرة عامّة على كل الخطوات.
2. **أنتج الفيديو** وفق `01-demo-script.md` (2–3 أيّام عمل).
3. **جهّز الـ repo** حسب قائمة Pre-Launch في الـ checklist.
4. **يوم النشر**: انسخ من `02-launch-copy.md` حسب المنصّة.

## نقطة الاسترجاع

قبل أي تعديل، تمّ إنشاء tag:
```bash
git tag -l -n1
# v1.0.0-baseline  Restore point before launch preparation

# للعودة للحالة الأصليّة:
git reset --hard v1.0.0-baseline
```

## الوقت المتوقَّع لكل مرحلة

| المرحلة | الوقت |
|---|---|
| تسجيل الفيديو | 1 يوم |
| مونتاج + تلميع | 1 يوم |
| تجهيز الـ repo (GIF, release, social preview) | نصف يوم |
| اختبار install.sh/ps1 على جهاز نظيف | نصف يوم |
| النشر + اليوم الأوّل | يوم كامل (8+ ساعات) |
| متابعة يوم 2–3 | نصف يوم/يوم |
| **الإجمالي** | **4–5 أيّام عمل** |

## الأولويّة القصوى

**الفيديو**. إذا الفيديو ضعيف، النشر يفشل بغضّ النظر عن المحتوى. خصّص له وقتاً.

## عبارة واحدة للتذكير

> "الكود جاهز. الـ launch copy جاهز. الخطّة جاهزة. **الباقي أنت**."
