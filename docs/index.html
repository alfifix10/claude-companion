أسئلة شائعة

هل أحتاج اشتراك Claude Max؟

نعم. الإضافة تعمل عبر اشتراكك الشخصيّ في Claude Max — لا تشتري شيئاً إضافيّاً ولا تدفع مقابل API.

هل هي مجّانيّة؟

نعم، مجّانيّة في هذه المرحلة. تحتاج فقط اشتراك Max الخاصّ بك.

هل بياناتي آمنة؟

كلّ شيء يعمل محليّاً على جهازك عبر Claude CLI — لا خوادم وسيطة. وضع Pro يطلب موافقتك قبل أيّ إجراء يُعدّل جهازك.

أيّ المتصفّحات مدعومة؟

كلّ متصفّحات Chromium: Chrome · Brave · Edge · Opera · Vivaldi · Arc.

لماذا التثبيت ليس بنقرة من المتجر؟

الإضافة تحتاج خادماً محليّاً للتواصل مع Claude CLI، وهذا خارج نطاق متجر الإضافات حاليّاً. المثبِّت يختصر الإعداد إلى نقرة مزدوجة.
